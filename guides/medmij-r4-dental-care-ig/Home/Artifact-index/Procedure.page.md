The mz-Procedure profile is largely based on the [nl-core-Procedure-event](https://simplifier.net/packages/nictiz.fhir.nl.r4.nl-core/0.12.0-beta.1/files/2885832) profile, but not derived from the latter, as the underlying functional requirements are not entirely compatible. An *extensible* binding to the ProcedureTypeDentalCare value set has been added on `.code`.
